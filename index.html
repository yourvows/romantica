Дорогая Муфассал

Поздравляю тебя с днём рождения. Желаю чтобы ты оставалась такой же прекрасной как сейчас. Была счастлива несмотря ни на что, желаю тебе крепкого здоровья и успехов в жизни. Также хочу сказать, что ты самый прекрасный человек в моей жизни. Не знаю почему так хочется давать знании, отвечать на твои вопросы, помочь тебе развиться и чтобы у тебя было всё было прекрасно, и не знаю почему сердце так ищет тебя, скучает и не хочет отдавать тебя никому. Ты самый дорогой мне человек под которому я всегда стараюсь адаптироваться и дать всё возможное. Ты ко многому меня научила: ждать, надеется, быть независимым от своих мыслей, любить по-настоящему, переживать и даже очароваться лишь смотрев на фото любимого человека. Ты дала мне шанс стать лучше чем вчера и благодаря тебе я начал читать книги и изучать нашу религию. Ты просто не знаешь как я хочу дотронуться за эти щёчки, пухленькие, самые любимые на этом свете для меня. Ты тот самый человек который может заставить меня измениться. Я благодарен, что ты существуешь в то время, который я живу.

С любовью, Иброхим

  • Languages Used

    This page was written in HTML and CSS. The CSS was compiled from SASS. I used Normalize as my CSS reset and -prefix-free to save myself some headaches. I haven't quite gotten the hang of Slim for compiling into HTML, but someday I'll use it since its syntax compliments that of SASS. Regardless, this could all be done in plain HTML and CSS.

  • How it Works

    Using the sibling and checked selectors, we can determine the styling of sibling elements based on the checked state of the checkbox input element. One use, as demonstrated here, is an entirely CSS and HTML accordion element. Media queries are used to make the element responsive to different screen sizes.

  • Points of Interest

    By making the open state default for when :checked isn't detected, we can make this system accessable for browsers that don't recognize :checked. The fallback is simply an open accordion. The accordion can be manipulated with Javascript (if needed) by changing the "checked" property of the input element.
